# MartialArtsDBApp
A web application designed to bridge the gap between martial arts schools and prospective students. Developed as a minimum viable prototype with assistance from ChatGPT, the app provides a comprehensive platform for exploring martial arts culture and connecting with schools.

**Original idea and design:** Alexander Dave Flores Respicio
**Code assistance:** ChatGPT

## Key Features
- **Martial Arts Styles Catalog:** Explore different martial arts disciplines and their unique characteristics.
- **School Database:** Access a directory of martial arts schools across Canada.
- **Service Inquiry Form:** Allows school owners to request digital marketing support.
- **Media Catalog:** Browse video games, shows, and movies with strong martial arts themes.
- **Documentary & News Listings:** Discover documentaries and news stories highlighting martial artsâ€™ role in character development and self-defense.

## Supporting Python Scripts
- `gym_profile_maker.py`: Generate JSON objects from martial arts school information.
- `location_pair_generator.py`: Create listings of province-city pairs for geographic reference.
- `youtube_info_collector.py`: Extract and structure metadata from YouTube URLs.
- `youtube_video_searcher.py`: Search YouTube for martial arts-related news content.

## Purpose & Impact
This project demonstrates a practical application of web development, data management, and media integration to support martial arts education and community engagement.
